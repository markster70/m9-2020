Objectives

From 2014 to 2016, M9 Digital worked with Barclays Bank on a major change project to re-build the offering to their Wealth Customers.

We we asked to partner with the UX / UI teams assigned to the project in order to create the code for non-trading aspects of the site, following the design requirements provided by the Smart Investor Team.

The key technical objectives were to use Barclays prescribed Front-End Frameworks ( Twitter Bootstrap, JQuery, AngularJS ) as well as other modern development technologies, and create a bespoke HTML / CSS framework

Outcomes

From 2014 to 2016, M9 Digital worked with Barclays Bank on a major change project to re-build the offering to their Wealth Customers.

We delivered a front-end framework based on the UI / Ux work, and created the necessary components in Angular JS to make a re-usable and scalable architecture

We also delivered a pattern library for future use by the JavaScript engineers that were working on the project
